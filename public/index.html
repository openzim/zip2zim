What's else can I do?

There are various ways you can customise your Zim files:

  • Put an favicon.png file in the zip
  • Add config.json
  • The following properties are supported in config.json
    • title
    • name
    • favicon
    • description
    • language
    • index
    • creator
    • publisher

About

This project was worked on at the Kiwix Wikimania 2016 Hackathon as an effort to easily generate Zim files from PhET Simulations
The project is written in NodeJS and is hosted on a Digital Ocean Droplet
